# ETL - Helppi Analytics Case

## Fonte dos dados
Dados fictícios gerados via Python com a biblioteca Faker.

## Etapas do ETL
1. Leitura dos dados brutos (CSV)
2. Padronização de tipos e datas
3. Criação de métricas derivadas (SLA, tempo)
4. Modelagem dimensional (dimensões e fatos)
5. Persistência dos dados tratados

## Saída
Arquivos CSV prontos para consumo em Power BI e SQL.
