# Helppi Analytics Case

Projeto de case analítico end-to-end simulando um cenário real de BI para a empresa Helppi.

O projeto cobre todas as etapas do pipeline de dados:
- Geração de dados fictícios
- ETL em Python
- Modelagem e views analíticas em MySQL
- Consumo e visualização no Power BI
- Documentação técnica completa
